# Library Management System (Java)

A simple console-based Library Management System built using Java. This project allows users to manage books by adding, issuing, returning, and viewing them.

---

## Features

*  Add new books
*  Issue books
*  Return books
*  View all books
*  Tracks book availability status

---

##  Technologies Used

* Java
* ArrayList (Java Collections Framework)
* OOP Concepts (Classes & Objects)

---

## Usage

After running the program, you will see a menu:

```
1. Add Book
2. Issue Book
3. Return Book
4. View All Books
5. Exit
```

* Select an option by entering the corresponding number.
* Follow the prompts to perform operations.

## Limitations

* Data is not stored permanently (no database)
* Console-based interface only
* No user authentication
